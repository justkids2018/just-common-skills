## 拆分原则

- 每个 Task **≤ 3 个文件**
- 每个 Task **≤ 200 行 diff**
- 每个 Task **可独立验证**（编译过 / 测试过 / 手测过）
- 每个 Task **可独立 commit**
- Task 之间**有清晰的顺序依赖**
