# 数据铁律与分级标准

> 研究质量的上限，取决于数据源的质量。  
> **核心铁律：宁可空格，不可造数。**

---

## 数据三级分类

所有研究信息按可信度分为三级，使用时须明确标注：

| 级别 | 来源类型 | 用途 | 标注要求 |
|------|---------|------|---------|
| **A 级**（唯一财务来源） | SEC EDGAR、公司 IR 官网、港交所、监管机构官网 | 财务数字唯一来源 | 直接引用，注明文件名和日期 |
| **B 级**（辅助参考） | 机构研究报告、电话会实录、Press Release、学术论文（NBER/SSRN） | 行业判断和指引参考 | 注明机构名称、发布日期 |
| **C 级**（情绪参考） | KOL 观点、社交媒体、新闻、分析师转述、播客、YouTube | 情绪与视角参考 | 标注 ⚠️，严禁作为财务数据依据 |

---

## 核心使用规则

1. **财务数据**（营收、利润、债务等）**只能来自 A 级来源**，不得从 B/C 级中摘录数字
2. **B/C 级数据若引用财务数字**，须回溯至 A 级来源验证后方可使用
3. **无法核实的数字**一律标注 `[待官方核实]`
4. **HTML 报告中**，C 级来源的数字需加 ⚠️ 标识
5. **直接引用**（管理层原话）与**推断**（分析师解读）必须严格区分

## 数据来源速查

### A 级 — 公司财务（唯一可信来源）

| 来源 | 地址 | 用途 |
|------|------|------|
| **SEC EDGAR** | edgar.sec.gov | 美股 10-K / 10-Q / 8-K / DEF 14A（管理层薪酬）全文检索 |
| **公司 IR 官网** | 各公司投资者关系页面 | 财报 PDF、业绩电话会议记录、投资者 Day 演示 |
| **港交所** | hkexnews.hk | 港股年报、季报、公告 |
| **FINRA BrokerCheck** | finra.org | 经纪商合规记录、监管处罚 |

### A 级 — 宏观数据

| 来源 | 地址 | 用途 |
|------|------|------|
| **FRED (St. Louis Fed)** | fred.stlouisfed.org | 利率、GDP、失业、通胀、消费者信心等数百个宏观指标 |
| **BLS** | bls.gov | 就业数据、CPI 详细分项 |
| **Census Bureau** | census.gov | 人口结构、收入分配、消费数据 |
| **BIS** | bis.org | 全球银行业/金融体系稳定性报告 |
| **IMF** | imf.org | 全球经济展望、各国债务数据 |
| **World Bank** | data.worldbank.org | 新兴市场数据、全球发展指标 |

### A/B 级 — 学术与政策研究

| 来源 | 地址 | 用途 |
|------|------|------|
| **NBER** | nber.org | 顶级经济学研究工作论文，先于期刊发表 |
| **SSRN** | ssrn.com | 金融/法律/经济学预印本，免费获取 |
| **CRS（国会研究服务）** | crsreports.congress.gov | 美国国会的行业政策研究报告，覆盖监管风险 |
| **GAO（政府问责局）** | gao.gov | 政府项目效率审查，找监管落地风险 |

## 研究工作流中的数据源使用顺序

```
快速研究（/just-value-red-publish 标准流程）
├── Phase 1 基础财务     → SEC EDGAR / 公司 IR 官网（A级）
├── Phase 1 舆情搜索     → X / Reddit / YouTube（C级，⚠️标注）
├── Phase 1 机构报告     → ARK / Oaktree 等（B级）
├── Phase 1 宏观数据     → FRED / IMF + 宏观四维框架判断
└── Phase 2 分析        → 财务数字仅用 A级，观点引用标注来源级别

完整研究（完整研究类型）
├── Step 1 监管/学术    → SEC / NBER / SSRN / CRS / GAO（A/B级）
├── Step 2 深度文章     → Substack / Seeking Alpha（B/C级）
├── Step 2.5 KOL观点   → X / Reddit / Substack（C级，⚠️标注）
├── Step 3 商业分析     → 综合A+B+C级，财务数字仅用A级
├── Step 4 竞争格局     → 各公司 SEC EDGAR 交叉验证（A级）
└── Step 4.5 宏观压测  → FRED + BIS + IMF + 行业报告（A/B级）
```

