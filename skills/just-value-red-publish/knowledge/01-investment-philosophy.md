# 投资核心哲学

> 本文档定义价值投资的核心理念、决策框架与跟踪原则。  
> 这是整套研究体系的顶层逻辑，每次研究公司前先审视本框架。  
> 更新本文档时无需修改 SKILL.md 工作流。

---

## 什么是价值投资

价值投资不只是财务数字的比较，更是**认知层面的竞争**。  
一笔好的投资，本质是在市场定价错误的地方，用更准确的认知做出更正确的判断。

### 三个核心问题

研究体系必须回答三个问题，才算完整：

1. **这个时代，什么公司值得长期持有？**（时代认知层）
2. **这家公司，是否经得起宏观结构的压力测试？**（宏观过滤层）
3. **在正确价格上，如何做出买入决策？**（估值决策层）

## 核心财务判断维度

| 维度 | 核心指标 | 为什么 |
|------|---------|--------|
| **现金流** | 自由现金流（FCF） | 真实的赚钱能力，无法造假 |
| **估值** | P/FCF、EV/EBITDA、DCF | 再好的公司买贵了也亏钱 |
| **成长** | 营收增速、用户增速、市场份额变化 | 决定未来盈利能力的上限 |
| **负债** | 净现金 / 有息债务、现金覆盖倍数 | 决定公司能否活过周期 |

> **关于 EPS**：EPS（每股净利润）仅作辅助参考，不作核心判断依据。  
> 折旧方式、股票回购、一次性收益均可影响 EPS，而不改变真实现金创造能力。

## 快速研究检查清单

每次研究公司时对照执行：

```
第零层 ── 时代定位（一票否决）
  □ 这家公司是重写者还是被重写者？
  □ 若被重写：高风险（1-3年）/ 中风险（3-7年）/ 低风险（7年+）？
  → 被重写且高风险 → 停止，跳过

第一层 ── 宏观四维过滤
  □ 区域本土优先：受益 / 中性 / 受损
  □ 反脆弱性：反脆弱 / 强韧 / 脆弱
  □ 消费分层：顺势 / 中性 / 逆势
  □ AI时代冲击：卡位受益 / 工具化存活 / 被瓦解风险
  → 负面评级数量：___个（影响宏观调整系数）

第二层 ── 公司质量四维
  □ 护城河：深厚 / 一般 / 存疑
  □ 管理层：优秀 / 合格 / 存疑
  □ 竞争格局：有利 / 中性 / 危险
  □ 财务健康：健康 / 中性 / 警示
  → 公司质量综合：优秀 / 合格 / 存疑

第三层 ── 估值与概率赔率
  □ 合理估值（至少两种方法）：$___
  □ 当前价格：$___，折扣：___折
  □ 宏观调整后买入价：$___
  □ 三情景期望收益：___% （≥15%？）
  □ 赔率：___:1 （≥2:1？）
  → 期望收益或赔率不达标 → 停止，跳过

最终决策
  □ 芒格逆向三问已完成？
  □ 最终结论：买入 / 观望 / 跳过 / 触发深度研究
  □ 建议仓位：___% （凯利 × 1/4~1/2）
  □ 触发重评条件已写明？
```

## 最终决策机制

### 综合评分表

| 层级 | 评估项 | 评级选项 | 否决权 |
|------|--------|---------|--------|
| 第零层 | 时代定位（重写者/被重写者） | — | **一票否决** |
| 第一层 | 区域本土优先 | 受益 / 中性 / 受损 | — |
| 第一层 | 反脆弱性 | 反脆弱 / 强韧 / 脆弱 | — |
| 第一层 | 消费分层 | 顺势 / 中性 / 逆势 | — |
| 第一层 | AI时代冲击 | 卡位受益 / 工具化 / 被瓦解 | — |
| 第二层 | 公司质量综合 | 优秀 / 合格 / 存疑 | — |
| 第三层 | 期望收益 | ≥15% / 不足 | **一票否决** |
| 第三层 | 赔率 | ≥3:1 / 2-3:1 / <2:1 | **一票否决** |

### 决策矩阵

```
【硬性否决】以下任意一条成立 → 直接跳过，停止分析：
  × 第零层：公司正在被重写（非重写者）且高风险（1-3年）
  × 期望收益 < 15%/年
  × 赔率 < 2:1

【建议买入】全部条件满足：
  ✓ 通过所有硬性否决
  ✓ 第一层负面评级 ≤ 1个，且"受损"与"脆弱"不同时出现
  ✓ 第二层公司质量：优秀或合格
  ✓ 期望收益 ≥ 20%/年，赔率 ≥ 3:1
  → 买入，仓位 = 凯利系数 × 1/4 到 1/2

【建议观望】：
  ✓ 通过所有硬性否决
  ✓ 期望收益 15-20% 之间，或赔率 2-3:1
  ✓ 或第一层存在 1-2 个负面评级
  → 观望：等价格下跌或宏观信号改善后重新评估

【触发深度研究】：
  ✓ 公司质量优秀，但宏观存在重大不确定性
  ✓ 或三种情景概率分歧过大（最大分歧 > 30%）
  → 先做深度研究，再给最终结论
```

### 最终输出模板（强制执行，不可跳过）

```
【决策结论】买入 / 观望 / 跳过 / 触发深度研究

【一句话理由】_______________________（20字以内）

【核心风险】_______________________（最可能导致判断错误的单一因素）

【触发重评条件】
  升级买入：_______________________（正面信号）
  降级跳过：_______________________（负面信号）

【建议仓位】____%（仅"买入"结论时填写）
```

## 关键跟踪指标原则

每家公司研究结束后，必须输出 **2 个最关键的跟踪指标**：

- 指标必须可量化，有具体当前值
- 必须给出"达到"和"未达到"的**双向意义**（不只写正面情景）
- 达到目标值 → 验证核心投资论点
- 未达到目标值 → 触发重新评估买入逻辑

**选取原则**：
- 选择最能**证伪**"核心论点"的指标，而不是最容易达成的
- 优先选择**领先指标**（预判未来），而非滞后指标（确认已知）
