# 宏观四维分析框架

> 宏观过滤是公司研究的**前置筛选**，不是附属。  
> 一家财务优秀的公司，如果卡在错误的宏观趋势里，也可能是陷阱。  
> 四个维度每个都必须有明确结论，不允许"有待观察"。

---

## 第零层：AI 时代定位（一票否决层）

> 这是整个框架的认知前提。任何公司研究都必须先过这一关：  
> **在这个时代，这家公司是在被重写，还是在重写别人？**

### 转变一：结构性瓦解 — 中间人正在消失

AI 自动化正在系统性消灭依赖"信息差"和"流程中间环节"的商业模式。

被瓦解的典型角色：
- 信息聚合中间人（靠信息不对称收费）
- 流程管理角色（事务性执行中间层）
- 低附加值的人工审核 / 客服 / 销售环节

**关键问题**：这家公司的核心收入来源，有多少依赖"我知道你不知道"或"我中转你不中转"？

瓦解速度判断：
- 高风险（1-3年）：纯信息中介、标准化流程执行、低复杂度专业服务
- 中风险（3-7年）：复杂但可标准化的专业判断、规模化客户服务
- 低风险（7年+）：需要真实关系信任、高度定制化、创意类

---

### 转变二：结构性机遇 — AI 原生商业模式崛起

真正的 AI 原生公司不是"给旧产品加 AI 功能"，而是从底层重新设计商业逻辑。

三类 AI 原生机会：
1. **AI 原生品牌**：在 AI 时代建立新的用户信任关系
2. **创始人经济**：个人/小团队借助 AI 杠杆，输出过去需要百人团队的价值
3. **自主型公司（Autonomous Company）**：AI Agent 驱动，边际成本趋近于零

**关键问题**：这家公司在用 AI **降低成本**（防御性），还是在用 AI **重新定义产品边界**（进攻性）？

---

### 转变三：结构性瓶颈 — 新的真正护城河

AI 时代的护城河不再主要是"规模优势"或"网络效应"，而是三类稀缺资源：

| 瓶颈类型 | 核心稀缺性 | 典型例子 |
|----------|-----------|---------|
| **算力/能源** | 物理层面的稀缺，难以快速复制 | 数据中心、电网基础设施、芯片 |
| **界面与交互** | 用户习惯和信任的入口，一旦建立极难迁移 | 操作系统级 AI 助手、垂直领域超级 App |
| **信任与品牌** | 高风险决策场景中人类仍需要"我信任这个来源" | 权威数据库、监管背书的平台 |

**关键问题**：这家公司的护城河，落在哪个瓶颈上？还是一个都没卡位？

## 第一层：宏观四维过滤

### 维度一：区域本土优先

**核心判断**：  
全球正在从"无限互联互通"转向"区域化、本土化、主权化"。表现为贸易保护主义、友岸外包、数据本地化法规、金融监管碎片化（SWIFT 去依赖、跨境资本管制强化）。

**评估问题**：
- 公司营收 / 成本 / 用户 / 数据，有多少依赖跨境流动？
- 在区域化贸易集团情景下，公司是受益（本土化竞争对手被削弱）还是受损（跨境业务受阻）？
- 监管主权化（各国独立清结算、数据本地化要求）对商业模式有何冲击？
- 公司的护城河，在区域本土优先环境下是加强还是被限制？

**打标**：`受益` / `中性` / `受损`
- 受益：业务本土化、地缘壁垒保护、逆全球化创造本土强者
- 中性：业务闭环在单一市场内，宏观趋势影响有限
- 受损：核心业务依赖跨境（数据、供应链、用户、资本流动）

**时间线判断**：  
当前为 10 年以上结构性大周期，应按永久性结构变化对待，而非"等待消退的临时风险"。
- 2024-2028：加速期，贸易摩擦和监管主权化快速落地
- 2028-2033：制度固化期，新的区域化贸易体系形成
- 2033 之后：取决于是否出现新的全球协调机制

### 维度二：反脆弱性

**什么是反脆弱性（Taleb 框架）**：
- 脆弱（Fragile）：受到冲击时，损失超线性放大 → 越波动越亏
- 强韧（Robust）：受到冲击时，基本维持原状 → 抵御波动
- 反脆弱（Antifragile）：受到冲击时，从中获益 → 越波动越强

**评估问题（五问）**：
```
Q1 市场崩溃：极端波动时，公司的核心营收是上升还是下降？
Q2 监管突变：重大监管变化时，是被直接淘汰还是因行业洗牌受益？
Q3 技术颠覆：AI/区块链等技术冲击，对公司护城河是威胁还是赋能？
Q4 宏观衰退：经济下行时，核心用户行为如何变化（减少消费还是转移消费）？
Q5 黑天鹅：最大竞争对手倒闭，公司能否快速承接并受益？
```

**打标**：`反脆弱` / `强韧` / `脆弱`
- 反脆弱：Q1/Q2/Q5 中至少两个答案为"受益"
- 强韧：大多数场景能维持，没有明显放大损失的机制
- 脆弱：存在单一事件可瓦解核心盈利的"原罪式"风险

### 维度三：消费分层

**核心判断**：  
技术加速 + 区域本土优先 + AI 替代劳动力 → 社会财富加速向两端集中，传统中产阶层承压。这一趋势深刻影响消费品牌、金融平台、零售业态和内容平台的用户结构。

**评估问题**：
- 公司的核心用户群，在财富分布上偏向哪一端？
- 若消费分层加剧，目标市场是扩大（顶端财富增速更快）还是萎缩（中间层收缩）？
- 公司是否需要在战略上做出方向选择：往高端走（Goldman Sachs 路线）还是往下沉走？
- 公司当前的产品定位，能否在消费分层中找到稳定立足点？

**打标**：`顺势` / `中性` / `逆势`
- 顺势：服务顶端（奢侈品 / 高端财富管理）或受益于下沉（高性价比刚需）
- 中性：用户群分布广泛，消费分层影响相互抵消
- 逆势：核心用户群是正在收缩的中产阶层，且无明显迁移路径

**早期预警信号（跟踪）**：
- 高端消费品（爱马仕 / LVMH 季报）是否仍强势增长
- 零售业中端品牌（梅西百货类）销售数据是否持续萎缩
- 消费信贷违约率分布（顶端 vs 中下层的分化程度）
- 劳动力市场：高技能 vs 低技能工资增速差
- 储蓄率分布：高收入群体储蓄率 vs 中低收入群体储蓄率

### 维度四：AI 时代冲击

**评估问题（四问）**：
```
Q1 瓦解风险：公司核心业务是否依赖"信息差"或"流程中间人"？
   → 是 = 高风险，AI Agent 会直接绕过

Q2 机遇捕获：公司是在构建 AI 原生产品，还是在给旧产品贴 AI 标签？
   → 真正 AI 原生 vs 表面 AI 化

Q3 瓶颈卡位：公司是否占据算力/界面/信任三个瓶颈之一？
   → 占据 = 真护城河；否则护城河可能是幻觉

Q4 Agent 适配：当 AI Agent 成为主要用户时，公司的产品是被调用还是被替代？
   → API 化程度、数据壁垒、品牌信任是否构成护城河
```

**打标**：`卡位受益` / `工具化存活` / `被瓦解风险`
- 卡位受益：占据至少一个瓶颈，且 Agent 时代会增加对该公司的调用
- 工具化存活：不占瓶颈，但业务在 Agent 时代仍有需求，只是利润率受压
- 被瓦解风险：核心业务是信息中间人，或产品不具备 API/数据壁垒

**行业瓦解速度分层**：
```
高风险（3年内实质影响）：
  - 信息聚合中间人：内容分发、广告中介、基础数据服务商
  - 标准化流程执行：低复杂度客服、文书处理、基础合规
  - 入门级专业服务：基础法律文书、基础财务分析报告

中风险（3-7年过渡期）：
  - 复杂专业判断：中级咨询、辅助诊断、市场研究
  - 规模化销售：B2B SaaS 的人工销售层

低风险（7年以上）：
  - 深度关系信任：私人银行、高端法律、家族办公室
  - 实体操作不可替代：精密外科、现场基础设施维护
  - 高度定制创意：顶级设计、文化内容创作

基本不被瓦解（护城河可能反而加强）：
  - 算力/能源基础设施：数据中心、电网、芯片
  - AI 本身的监管与信任背书机构
```
