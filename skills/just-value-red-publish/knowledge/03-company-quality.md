# 公司质量四维评估

> 宏观过滤通过后，进入传统的公司基本面研究。  
> 这一层的落脚点是：**这家公司本身是否优秀？**

---

## 维度一：商业模式与护城河

**核心追问**：
- 增长飞轮是否存在且自我强化？（用数据验证，不是逻辑上"应该存在"）
- 护城河类型识别：网络效应 / 转换成本 / 规模优势 / 品牌 / 数据 / 监管壁垒
- 在 AI 时代，现有护城河是加强还是削弱？（联动 knowledge/02-macro-framework.md §第零层）
- 商业模式是否存在"原罪式"风险（核心依赖某个可能被颠覆的单一机制）？

**飞轮拆解格式**（商业分析时使用）：
```
① [起点] → ② [放大机制] → ③ [自我强化步骤] → ① 循环
```

**打标**：`护城河深厚` / `护城河一般` / `护城河存疑`
- 深厚：至少两类护城河叠加，且 AI 时代加强
- 一般：存在一类护城河，但 AI 时代可能削弱
- 存疑：依赖单一机制，或存在"原罪式"风险

---

## 维度二：管理层与执行力

**核心追问**：
- 创始人 / CEO 的历史承诺与兑现记录（而非只看愿景）
- 激励机制是否与长期股东利益对齐（持股结构、薪酬设计）
- 在公司面临逆境时，管理层的行动模式是什么？

**数据来源**：
- SEC DEF 14A（管理层薪酬结构）
- 历届财报电话会指引 vs 实际兑现率
- 重大逆境时的公开表态与后续行动

**打标**：`管理层优秀` / `管理层合格` / `管理层存疑`
- 优秀：历史兑现率高，激励对齐，逆境中有明确执行力记录
- 合格：基本面没问题，但缺乏极端考验下的数据支撑
- 存疑：承诺与兑现存在明显落差，或激励机制与股东利益错位

---

## 维度三：行业竞争格局

**核心追问**：
- 行业是在集中（强者愈强）还是在碎片化（护城河难以建立）？
- 最危险的未来竞争者是谁？时间线在哪里？
- 公司在行业中的"关键决战时间窗口"是什么？

**竞争格局横向对比表格式**（行业分析时使用）：

| 公司 | 市值 | 营收 | 毛利率 | 核心 KPI | 护城河类型 |
|------|------|------|-------|---------|-----------|
| [标的公司] | | | | | |
| [竞争对手1] | | | | | |
| [竞争对手2] | | | | | |

**打标**：`格局有利` / `格局中性` / `格局危险`
- 有利：行业集中度提升，公司处于领先位置，3年内无强力颠覆者
- 中性：竞争激烈但相对稳定，公司有一席之地
- 危险：行业碎片化或存在明确的 3年内高威胁竞争者

---

## 维度四：财务健康度

**核心追问**：
- 真实的赚钱能力（自由现金流，而非利润表数字）
- 破产安全边际（现金覆盖债务倍数 + 极端场景维持年限）
- 资本效率（ROE / ROIC 是否在改善）

**关键指标**：

| 指标 | 健康标准 | 警示标准 |
|------|---------|---------|
| FCF | 为正且增长 | 持续为负 |
| 净现金/有息债务 | 净现金 > 有息债务 2 倍 | 高杠杆或隐性负债 |
| ROIC | 持续改善，高于资本成本 | 低于资本成本或恶化 |
| 利润率趋势 | 扩张或维持 | 持续收窄 |

**打标**：`财务健康` / `财务中性` / `财务警示`
- 健康：FCF 为正且增长，现金可覆盖债务 2 倍以上，ROIC 持续改善
- 中性：盈利但 FCF 波动，或高增速阶段暂时亏损但路径清晰
- 警示：FCF 持续为负，高杠杆，或存在隐性负债风险

## 第二层综合结论

```
护城河：_____ | 管理层：_____ | 竞争格局：_____ | 财务：_____

公司质量综合评级：
  优秀 → 四项中至少 3 项正面，无"存疑"/"警示"
  合格 → 多数正面，至多 1 项存疑，但核心商业模式稳固
  存疑 → 存在 2 项以上负面，或护城河/财务任一项为警示级
```

## 芒格逆向三问（买入前强制执行）

```
死亡问题：若此公司 5年内失败，最可能的死法是什么？
          → 若这个死法的概率 > 20%，重新审视买入逻辑

同步崩塌：若所有支持买入的论点同时错了，会发生什么？
          → 若结果是灾难性的，安全边际需要更大

行业消亡：此行业 10年后还存在吗？如果不存在，谁杀死了它？
          → 即使公司现在好，行业消亡风险必须定价
```
