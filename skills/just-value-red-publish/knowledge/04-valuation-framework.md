# 估值决策框架

> 好公司不等于好投资，关键是在合理价格上买。  
> **价格决定回报，质量决定安全边际。**

---

## 基础安全边际原则

巴菲特 / 芒格的核心：以低于内在价值 30-40% 的价格买入，为判断失误留余地。

```
合理估值方法（至少用两种交叉验证）：
  - P/E：适用于盈利稳定的成熟公司
  - EV/EBITDA：适用于资本结构复杂的公司
  - P/S：适用于高增速但尚未稳定盈利的成长公司
  - P/FCF：适用于现金流导向的价值评估
  - DCF：适用于现金流可预期的公司（须明确假设条件）

基础买入折扣：合理估值 × 7折（安全边际 40%）
关注区间：  合理估值 × 8折（安全边际 30%）
```

---

## 宏观调整系数

基准买入价 = 合理估值 × 7折

根据宏观四维评级（见 knowledge/02-macro-framework.md），调整折扣：

```
每出现一个负面评级（受损 / 脆弱 / 逆势 / 被瓦解风险）：
  → 额外再打 9折（要求更大折扣）

每出现一个正面评级（受益 / 反脆弱 / 顺势 / 卡位受益）：
  → 可放宽到 7折（安全边际要求降低）

示例：
  四维全负 → 合理估值 × 7折 × 0.9^4 ≈ 合理估值 × 4.6折
  四维全正 → 合理估值 × 8折
  两负两正 → 合理估值 × 7折 × 0.9^2 ≈ 合理估值 × 5.7折
```

> 注意：宏观调整系数不是精确公式，而是思维框架——它强制你在定价中纳入宏观判断，
> 避免"公司很好但时代不对"的陷阱。

## 概率与赔率框架

> 好的投资不只是"这家公司好"，而是"在这个价格上，期望值为正"。

### 第一步：写出三种情景

| 情景 | 概率（加总 100%） | 5年后目标价 | 对应年化收益 |
|------|----------------|-----------|------------|
| 牛市情景（最好结果） | P_bull = ?% | $? | ?% |
| 基准情景（最可能结果） | P_base = ?% | $? | ?% |
| 熊市情景（最差结果） | P_bear = ?% | $? | ?% |

### 第二步：计算期望年化收益

```
期望收益 = P_bull × 牛市年化 + P_base × 基准年化 + P_bear × 熊市年化

及格线：期望收益 ≥ 15%/年（否则风险补偿不足）
```

### 第三步：赔率检验（上行/下行比）

```
上行空间 = 牛市目标价 / 当前价 - 1
下行空间 = 1 - 熊市目标价 / 当前价

赔率 = 上行空间 / 下行空间

赔率标准：
  ≥ 3:1  → 赔率优秀，可以承受较低概率的成功
  2-3:1  → 赔率合格，需要基准情景概率 > 50%
  < 2:1  → 赔率不足，除非确定性极高否则不买
```

### 第四步：凯利系数参考（仓位上限）

```
凯利比例 = (期望收益 - 无风险利率) / 上行空间²

注意：凯利是理论上限，实际仓位取凯利比例的 1/4 到 1/2
→ 防止过度集中的安全边界
```

> 如果期望收益不及格，或赔率低于 2:1，即使公司质量很高也不应买入。
