📖 先搞清楚这是什么
Claude Code 是 Anthropic 做的 AI 编程工具，但它跟你用过的 AI 聊天工具有本质不同。

普通 AI 助手 = 聪明的顾问
你问一句，它回一句，给你建议，完事。

Claude Code = 会干活的员工
你说"帮我优化这段代码"，它自己去读文件、改代码、跑测试、提交 Git……一套连续动作打完，不用你一步步盯着。

💡 它不是模拟操作，是真实地在你电脑上读写文件、执行命令。你跟它说话，它就真的帮你干活。
然后，它的源码通过一个发布失误，两次被完整泄露出来——我们因此得以窥见一个亿级产品的内核长什么样。
