# 01 Positioning

## Mission

Build a reusable Agent foundation for project technical architecture and engineering workflow execution.

## Target Users

- Platform maintainers
- Tech leads
- AI-assisted developers

## In Scope

- Shared baseline governance
- Skill routing and execution conventions
- Injection and integration workflow
- Quality and safety controls

## Out of Scope

- Project business requirement details
- Product feature specs
- Language/framework-specific coding guides beyond baseline

## Success Criteria

- New project integration is completed with one command.
- Shared rules remain consistent across projects.
- Skill execution quality is measurable and continuously improved.
