# System Platform Docs

## Purpose

This folder defines the system-level operating model for the shared Agent platform.
It focuses on platform governance, execution standards, and quality control.

## Scope

- Platform positioning and boundaries
- Agent operating model
- Skill quality gate
- Evaluation and feedback loop
- Versioned architecture records
- HTML output standards
- Quickstart command card

## Why This Path

Use `system-platform/` as the canonical platform contract directory.
It is explicit, stable, and easy for both humans and AI to locate.
